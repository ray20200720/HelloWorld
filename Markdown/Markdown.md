# 代碼標示

如果在一行文字內可用單個 \`代碼\` 來標示

例如: `Hello World` 

如果是一整段可以用3個 \`\`\` 代碼區塊 \`\`\`來顯示

例如: 

``` js
console.log(Hello World)
```
